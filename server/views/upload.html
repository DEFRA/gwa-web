Upload

Users can be added to the system for additional organisations by uploading a CSV file (.csv).

The file must contain a header row. The following columns are required (in this order): emailAddress, givenName, surname, officeLocation, phoneNumber.

officeLocation should map to one of the existing office locations (see reference data page). If no match is found the office location will be mapped to Unassigned.

phoneNumber should be the UK mobile phone number e.g. 07000111222 you want to subscribe. Phone numbers can also be formatted with the UK country code e.g. +447000111222. If the phone number is not the correct format the import will be rejected.
