Manage ALB data

ALBs can be enrolled into the system by uploading a dataset of their users for them.

If the data provided by the ALB is a replacement of the existing data, uploading the new data is all that is needed.

However, there might be occasions where reviewing the existing dataset for an ALB is required. This is possible by downloading the dataset.
