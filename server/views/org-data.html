Manage ALB data

ALBs can be enrolled into the system by uploading a dataset of their users. The file uploaded replaces any existing file.

If the ALB no longer wants to be enrolled into the system their data can be deleted.

There may be occasions where the ALB data needs to be reviewed, this is possible by downloading the dataset.

Delete ALB data

Existing ALB data can be deleted from the system.
